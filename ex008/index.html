Principais formatações

Negrito / Destaques

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase, tenho um termo em destaque unsado a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Entendendo

Perceba que o não te semântica, é não ter significado, visualmente não há diferença, mas para o html5 as tags que dizemos ser semântica tem significado.
Então para os nossos casos use a tag <strong> para destaques, ou seja, negrito e a tag <em> para ênfase, ou seja, itálico.
